1. fetch API
: 자바스크립트에서 기본으로 제공하는 HTTP 요청 함수
 별도 라이브러리 설치 없어도 사용 가능
 특징
  - 내장 API라서 설치 필요 없음
  - Promise 기반 동작
  - 응답 JSON으로 변환하려면 .JSON 직접 호출해야함
  - 에러 처리 따로 해줘야됨 (.then / .finally 등)

2. Axios
: HTTP 요청 더 쉽게 다루기위해 사용하는 외부 라이브러리
 특징
  - 설치 필요 (npm install axios)
  - 자동으로 JSON 변환
  - HTTP 에러 자동 처리
  - 요청 / 응답 인터셉터 사용 가능
  - 코드 간결
